# Tetris

Classic tetris on Pixi.js

See the [Game here](http://smival.com/tetris)

Typescript + pixi + webpack + hotreload

## Controls

move: arrows left, right;

rotate: space;

hard drop: arrow up;

soft drop: arrow down;

## How to use sources

```sh
npm install
```
run local server (http://localhost:8080/)
```sh
npm start
```
build project
```sh
npm run build
```
